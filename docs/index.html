ConfiguRateSDK

Requirements

  • iOS 9.0+
  • Xcode 8.1+
  • Swift 3.0+

Installation

CocoaPods

To integrate ConfiguRateSDK into your Xcode project using CocoaPods, add it to your Podfile:

source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '9.0'
use_frameworks!

target '<Your Target Name>' do
    pod 'ConfiguRateSDK'
end

Then, run the following command:

$ pod install

Usage

  1. Import ConfiguRateSDK framework header in your ViewController
///Swift
import ConfiguRateSDK
  1. Add the following to your view controller.
///Swift
let virtualId = "YOUR_VIRTUAL_ID" // Contact Appable (http://goappable.com) to get YOUR_VIRTUAL_ID
ConfiguRateButton(target: self, virtualId: virtualId)
